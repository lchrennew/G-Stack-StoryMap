User Story Mapping

## Dev Environment Prerequisitions
- Java 10+
- Neo4j
- Node.js
- Gulp
- Rollup
- Maven

## Run Environment Prerequisitions
- Java 10+
- Neo4j
- Chrome

## Features
- Multiple StoryMappings
- Flexible & Smart Card Drag&Drop
- Release Planning
- Card Description in Markdown
- Card Comments
- Card Acceptance Criteria
